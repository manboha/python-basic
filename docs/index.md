# 파이썬 기초 - 서문

---

## 이 책에 대하여

이 책은 **데이터 분석, 통계 분석, 인공지능** 등을 공부하려는 사람들을 위한 Python 기초 교재입니다. 

프로그래밍을 한 번도 해본 적이 없어도 괜찮습니다. 개발 환경 설정부터 시작해서 NumPy, Pandas 같은 데이터 분석 라이브러리를 사용하는 단계까지, 차근차근 안내합니다.

---

## 이 책의 구성

**0. 개발 환경 구축**  
VS Code 설치, Python 설치, 가상환경 설정, 첫 코드 실행까지 모든 준비 과정을 다룹니다.

**1. 파이썬 기초 문법**  
변수, 숫자, 문자열, 불린, 입출력 등 가장 기본적인 문법을 배웁니다.

**2. 데이터 타입과 자료구조**   
리스트, 튜플, 딕셔너리, 집합을 배우고 데이터를 효과적으로 관리하는 방법을 익힙니다.

**3. 제어문**   
조건문(if)과 반복문(for, while)으로 프로그램의 흐름을 제어합니다.

**4. 함수**   
코드를 재사용 가능한 함수로 만들고, 매개변수, 반환값, 스코프 등을 이해합니다.

**5. 라이브러리 사용법**   
NumPy와 Pandas의 기초를 배우며 본격적으로 데이터를 다루기 시작합니다.

**6. 파일 입출력**   
텍스트, CSV, JSON, Excel 파일을 읽고 쓰는 방법을 배우고 데이터 파이프라인을 구축합니다.

**7. 클래스 기초 (선택)**   
객체 지향 프로그래밍의 기초를 배우고, scikit-learn 같은 라이브러리의 API를 이해합니다.
