## 1.1 변수와 데이터의 시작

프로그램을 작성하는 것은 인간의 생각을 코드로 옮겨 담는 일입니다. 그 생각의 재료가 바로 **데이터(data)**입니다. 숫자를 계산하거나, 텍스트를 분석하거나, 이미지를 변환하는 모든 작업은 데이터를 다루는 것에서 시작합니다. 그런데 데이터는 끊임없이 변하고, 여러 곳에서 재사용되어야 합니다. 매번 같은 값을 반복해서 입력할 수는 없습니다. 이때 필요한 것이 **변수(variable)**입니다.

변수는 데이터를 담는 상자입니다. 상자에 이름표를 붙여서 나중에 쉽게 찾을 수 있게 만듭니다. 예를 들어 학생의 점수를 다룬다면, 점수를 매번 85라고 쓰는 대신 `score`라는 이름의 변수에 담아둡니다.

```python
score = 85
print(score)
```

출력:

```
85
```

여기서 `score`가 변수 이름이고, ` = `는 할당(assignment) 연산자입니다. "score라는 이름의 상자에 85라는 값을 넣는다"는 의미입니다. 수학의 등호(같다)와는 다른 의미라는 점을 주의해야 합니다.

변수는 언제든 새로운 값으로 바꿀 수 있습니다.

```python
score = 85
print(score)

score = 92
print(score)
```

출력:

```
85
92
```

처음에 85였던 값이 92로 바뀌었습니다. 기존 값은 사라지고 새 값으로 대체됩니다.

### 1.1.1 변수 이름 규칙

변수 이름은 자유롭게 정할 수 있지만, 몇 가지 규칙이 있습니다.

1. **영문자, 숫자, 밑줄(`_`)만 사용 가능**
    - 가능: `student_name`, `score1`, `_temp`
    - 불가능: `student-name` (하이픈 사용 불가), `2score` (숫자로 시작 불가)

2. **대소문자 구분**
    - `score`와 `Score`는 다른 변수입니다.

3. **예약어 사용 불가**
    - Python이 이미 쓰고 있는 단어는 변수명으로 쓸 수 없습니다.
    - 예: `if`, `for`, `while`, `class`, `def`, `import`, `True`, `False`, `None` 등

4. **관례: snake_case 사용**
    - 여러 단어를 연결할 때 밑줄로 구분합니다.
    - 좋은 예: `student_score`, `total_count`
    - 나쁜 예: `StudentScore` (이것은 클래스명 관례), `studentscore` (가독성 낮음)

### 1.1.2 동적 타이핑

Python은 변수를 선언할 때 타입(자료형)을 명시하지 않습니다. 같은 변수에 숫자를 넣었다가 나중에 문자열을 넣어도 됩니다.

```python
data = 100
print(data)

data = "백 점"
print(data)
```

출력:

```
100
백 점
```

이것을 **동적 타이핑(dynamic typing)** 이라고 합니다. 변수의 타입이 고정되지 않고, 담긴 값에 따라 자동으로 결정됩니다. 편리하지만, 실수로 다른 타입을 넣어서 오류가 생길 수 있으니 주의해야 합니다.

변수명을 정할 때 고민되나요? AI에게 물어보세요.

"나이를 저장하는 변수명 추천해줘. 총 5개 정도."

AI는 `age`, `user_age`, `current_age`, `person_age`, `age_value` 같은 옵션을 제시할 것입니다. 이 중에서 프로젝트 맥락에 맞는 것을 선택하면 됩니다.

## 1.2 숫자형 데이터

### 1.2.1 정수(int)와 실수(float)

Python에서 숫자는 크게 두 가지로 나뉩니다.

**정수(integer, int):** 소수점이 없는 숫자

```python
count = 10
temperature = -5
population = 1000000
```

**실수(floating-point, float):** 소수점이 있는 숫자

```python
price = 3.14
weight = 72.5
ratio = 0.95
```

변수에 담긴 값의 타입을 확인하려면 `type()` 함수를 사용합니다.

```python
a = 10
b = 3.14

print(type(a))
print(type(b))
```

출력:

```
<class 'int'>
<class 'float'>
```

정수와 실수를 함께 연산하면 결과는 자동으로 실수가 됩니다.

```python
result = 10 + 3.5
print(result)
print(type(result))
```

출력:

```
13.5
<class 'float'>
```

### 1.2.2 기본 산술 연산

Python은 기본 사칙연산을 지원합니다.

```python
a = 10
b = 3

print(a + b)   # 덧셈: 13
print(a - b)   # 뺄셈: 7
print(a * b)   # 곱셈: 30
print(a / b)   # 나눗셈: 3.3333...
```

**나눗셈의 특별한 경우:**

일반 나눗셈(`/`)은 항상 실수를 반환합니다. 정수끼리 나눠도 마찬가지입니다.

```python
print(10 / 2)   # 5.0 (실수)
```

몫만 필요하다면 `//` (버림 나눗셈)을 사용합니다.

```python
print(10 // 3)  # 3 (몫)
```

나머지만 필요하다면 `%` (나머지 연산자)를 사용합니다.

```python
print(10 % 3)   # 1 (나머지)
```

**거듭제곱:**

```python
print(2 ** 3)   # 2의 3승: 8
print(5 ** 2)   # 5의 2승: 25
```

**연산자 우선순위:**

수학과 동일하게 곱셈과 나눗셈이 덧셈과 뺄셈보다 먼저 계산됩니다. 괄호를 사용하면 우선순위를 변경할 수 있습니다.

```python
print(2 + 3 * 4)       # 14 (곱셈 먼저)
print((2 + 3) * 4)     # 20 (괄호 먼저)
```

### 1.2.3 숫자 타입 변환

문자열을 숫자로 바꾸거나, 실수를 정수로 바꿔야 할 때가 있습니다.

**문자열을 정수로:**

```python
text = "123"
number = int(text)
print(number + 10)  # 133
```

**문자열을 실수로:**

```python
text = "3.14"
number = float(text)
print(number * 2)   # 6.28
```

**실수를 정수로 (소수점 버림):**

```python
pi = 3.14
integer_pi = int(pi)
print(integer_pi)   # 3
```

**변환 실패 시 에러:**

숫자로 바꿀 수 없는 문자열을 변환하려고 하면 `ValueError`가 발생합니다.

```python
text = "abc"
number = int(text)  # ValueError: invalid literal for int() with base 10: 'abc'
```

이런 에러를 방지하려면 나중에 배울 예외 처리(`try-except`)를 사용합니다.

## 1.3 문자열(String)

### 1.3.1 문자열 생성과 출력

문자열은 텍스트 데이터를 표현합니다. 작은따옴표(`'`) 또는 큰따옴표(`"`)로 감싸서 만듭니다.

```python
name1 = 'Alice'
name2 = "Bob"
print(name1)
print(name2)
```

출력:

```
Alice
Bob
```

작은따옴표와 큰따옴표 중 어느 것을 선택해도 무방하지만, 문자열 내부에 특정 따옴표가 들어가는 경우 겉을 감싸는 것은 다른 종류를 사용합니다.

```python
message = "He said, 'Hello!'"
print(message)
```

출력:

```
He said, 'Hello!'
```

**여러 줄 문자열:**

삼중 따옴표(`"""` 또는 `'''`)를 사용하면 여러 줄에 걸친 문자열을 만들 수 있습니다.

```python
poem = """장미는 빨갛고
제비꽃은 파랗다"""

print(poem)
```

출력:

```
장미는 빨갛고
제비꽃은 파랗다
```

### 1.3.2 문자열 연산

**문자열 연결 (`+`):**

```python
first_name = "Kim"
last_name = "Minho"
full_name = first_name + " " + last_name
print(full_name)
```

출력:

```
Kim Minho
```

**문자열 반복 (`*`):**

```python
laugh = "ha" * 3
print(laugh)
```

출력:

```
hahaha
```

**주의: 문자열과 숫자는 직접 결합 불가**

```python
age = 25
message = "나이: " + age  # TypeError: can only concatenate str (not "int") to str
```

숫자를 문자열로 변환해야 합니다.

```python
age = 25
message = "나이: " + str(age)
print(message)
```

출력:

```
나이: 25
```

### 1.3.3 문자열 포맷팅

변수를 문자열에 삽입하는 방법은 여러 가지가 있습니다. 가장 권장되는 방식은 **f-string**입니다.

**f-string (Python 3.6+):**

문자열 앞에 `f`를 붙이고, 중괄호 `{}`안에 변수나 표현식을 넣습니다.

```python
name = "Alice"
age = 25
score = 92.5

message = f"이름: {name}, 나이: {age}, 점수: {score}"
print(message)
```

출력:

```
이름: Alice, 나이: 25, 점수: 92.5
```

중괄호 안에서 계산도 가능합니다.

```python
a = 10
b = 20
print(f"{a} + {b} = {a + b}")
```

출력:

```
10 + 20 = 30
```

**`.format()` 메서드:**

f-string이 나오기 전에 주로 쓰던 방식입니다.

```python
message = "이름: {}, 나이: {}".format("Bob", 30)
print(message)
```

출력:

```
이름: Bob, 나이: 30
```

**오래된 방식 (`%` 연산자):**

C 언어 스타일입니다. 현재는 거의 쓰지 않지만, 오래된 코드에서 볼 수 있습니다.

```python
message = "이름: %s, 나이: %d" % ("Charlie", 35)
print(message)
```

**권장:** f-string을 사용하세요. 가장 직관적이고 빠릅니다.

### 1.3.4 기본 문자열 메서드

문자열 객체는 다양한 메서드(함수)를 가지고 있습니다.

**대소문자 변환:**

```python
text = "Hello World"
print(text.upper())   # HELLO WORLD
print(text.lower())   # hello world
```

**공백 제거:**

```python
text = "  Python  "
print(text.strip())   # "Python" (양쪽 공백 제거)
```

**문자열 치환:**

```python
text = "I like apples"
new_text = text.replace("apples", "bananas")
print(new_text)   # I like bananas
```

**문자열 분할 (리스트로 변환):**

```python
text = "apple,banana,cherry"
fruits = text.split(",")
print(fruits)   # ['apple', 'banana', 'cherry']
```

`split()`은 문자열을 특정 구분자로 나눠서 리스트로 만듭니다. 리스트는 2장에서 자세히 다룹니다.

## 1.4 불린(Boolean)과 비교 연산

### 1.4.1 참과 거짓

불린(Boolean)은 참(`True`) 또는 거짓(`False`) 두 가지 값만 가지는 데이터 타입입니다. 조건을 판단할 때 사용됩니다.

```python
is_student = True
is_adult = False

print(type(is_student))   # <class 'bool'>
```

주의할 점은 대소문자입니다. `True`, `False`는 첫 글자가 대문자여야 합니다. `true`, `false`는 에러가 발생합니다.

### 1.4.2 비교 연산자

두 값을 비교하면 불린 결과가 나옵니다.

```python
a = 10
b = 20

print(a == b)   # False (같은가?)
print(a != b)   # True (다른가?)
print(a > b)    # False (a가 b보다 큰가?)
print(a < b)    # True (a가 b보다 작은가?)
print(a >= b)   # False (a가 b보다 크거나 같은가?)
print(a <= b)   # True (a가 b보다 작거나 같은가?)
```

**주의:** `=` vs `==`
- `=` : 할당 연산자 (값을 저장)
- `==` : 비교 연산자 (값이 같은지 확인)

```python
x = 5       # x에 5를 할당
x == 5      # x가 5인지 확인 (True)
```

**문자열 비교:**

문자열도 비교할 수 있습니다. 사전순으로 정렬됩니다.

```python
print("apple" < "banana")   # True (a가 b보다 앞)
print("apple" == "Apple")   # False (대소문자 구분)
```

### 1.4.3 논리 연산자

여러 조건을 결합할 때 사용합니다.

**`and`: 모두 참이어야 참**

```python
age = 25
has_license = True

can_drive = (age >= 18) and has_license
print(can_drive)   # True
```

**`or`: 하나라도 참이면 참**

```python
is_weekend = False
is_holiday = True

can_rest = is_weekend or is_holiday
print(can_rest)   # True
```

**`not`: 반대**

```python
is_raining = False
print(not is_raining)   # True
```

**복합 조건:**

```python
score = 85
attendance = 90

pass_course = (score >= 60) and (attendance >= 80)
print(pass_course)   # True
```

## 1.5 주석과 코드 가독성

### 1.5.1 주석의 종류

주석은 코드 실행에 영향을 주지 않는 설명문입니다. Python 인터프리터는 주석을 무시합니다.

**한 줄 주석:**

`#` 기호 뒤의 내용은 모두 주석입니다. 

```python
# 이것은 주석입니다
score = 85  # 변수에 점수 저장
```

**여러 줄 주석:**

삼중 따옴표를 사용합니다. 실제로는 문자열이지만 할당되지 않으면 아무 동작을 하지 않으므로 주석처럼 사용됩니다.

```python
"""
이것은 여러 줄 주석입니다.
함수나 모듈의 설명을 작성할 때 주로 사용합니다.
"""

score = 85
```

### 1.5.2 좋은 주석 vs 나쁜 주석

주석은 코드가 **무엇을** 하는지가 아니라 **왜** 그렇게 했는지를 설명해야 합니다.

**나쁜 주석 (코드를 그대로 반복):**

```python
# age 변수에 25를 할당
age = 25
```

코드만 봐도 알 수 있는 내용입니다.

**좋은 주석 (의도와 맥락 설명):**

```python
# 성인 기준 나이: 법적 기준 변경 시 이 값만 수정
ADULT_AGE = 18

# 테스트 케이스: 경계값 검증을 위해 17세 선택
age = 17
```

**더 좋은 방법: 코드 자체를 명확하게**

```python
# 나쁜 예
a = 18  # 나이 기준

# 좋은 예
adult_age_threshold = 18  # 주석 없이도 이해 가능
```

### 1.5.3 코드 스타일

Python 커뮤니티는 **PEP 8**이라는 코드 스타일 가이드를 따릅니다. 몇 가지 핵심 규칙:

- **들여쓰기는 스페이스 4칸**
- **연산자 앞뒤로 공백 하나**

```python
# 좋은 예
result = a + b

# 나쁜 예
result=a+b
```
  
- **한 줄 길이는 79자 이내 (권장)**
- **함수와 클래스 사이는 두 줄 띄우기**

VS Code의 Python 확장은 PEP 8 위반 사항을 자동으로 표시해줍니다.

**AI 활용 팁:**

코드를 작성한 후 AI에게 물어보세요.

"이 코드를 PEP 8 스타일로 수정해줘"

```python
# 수정 전
x=10
y=20
result=x+y
```

AI가 다음과 같이 수정해줄 것입니다.

```python
# 수정 후
x = 10
y = 20
result = x + y
```

## 1.6 입력과 출력

### 1.6.1 사용자 입력 받기

`input()` 함수는 사용자로부터 키보드 입력을 받습니다.

```python
name = input("이름을 입력하세요: ")
print(f"안녕하세요, {name}님!")
```

실행하면 프로그램이 멈추고 입력을 기다립니다. 사용자가 "Alice"를 입력하고 Enter를 누르면:

```
이름을 입력하세요: Alice
안녕하세요, Alice님!
```

**중요: `input()`은 항상 문자열을 반환합니다.**

숫자를 입력받아도 문자열로 저장됩니다.

```python
age = input("나이를 입력하세요: ")
print(type(age))   # <class 'str'>
```

숫자로 사용하려면 타입 변환이 필요합니다.

```python
age_str = input("나이를 입력하세요: ")
age = int(age_str)
print(f"10년 후 나이: {age + 10}")
```

### 1.6.2 출력 형식 제어

`print()` 함수는 여러 값을 한 번에 출력할 수 있습니다.

```python
name = "Alice"
age = 25
print("이름:", name, "나이:", age)
```

출력:

```
이름: Alice 나이: 25
```

**`sep` 매개변수: 구분자 지정**

### 1.7.1 요구사항

두 숫자를 입력받아 사칙연산 결과를 출력하는 프로그램을 만들어봅시다.

- 첫 번째 숫자 입력
- 두 번째 숫자 입력
- 덧셈, 뺄셈, 곱셈, 나눗셈 결과 출력
- 나눗셈은 소수점 2자리까지만 표시

### 1.7.2 코드 작성 과정

`calculator.py` 파일을 만들고 다음 코드를 작성합니다.

```python
# 간단한 계산기

# 사용자 입력 받기
num1_str = input("첫 번째 숫자를 입력하세요: ")
num2_str = input("두 번째 숫자를 입력하세요: ")

# 문자열을 실수로 변환
num1 = float(num1_str)
num2 = float(num2_str)

# 사칙연산
addition = num1 + num2
subtraction = num1 - num2
multiplication = num1 * num2
division = num1 / num2

# 결과 출력
print(f"\n--- 계산 결과 ---")
print(f"{num1} + {num2} = {addition}")
print(f"{num1} - {num2} = {subtraction}")
print(f"{num1} × {num2} = {multiplication}")
print(f"{num1} ÷ {num2} = {division:.2f}")
```

실행 예시:

```
첫 번째 숫자를 입력하세요: 10
두 번째 숫자를 입력하세요: 3

--- 계산 결과 ---
10.0 + 3.0 = 13.0
10.0 - 3.0 = 7.0
10.0 × 3.0 = 30.0
10.0 ÷ 3.0 = 3.33
```

**에러 처리 추가 (간단 버전):**

사용자가 숫자가 아닌 값을 입력하면 `ValueError`가 발생합니다. 이를 방지하기 위해 `try-except`를 사용할 수 있습니다. (예외 처리는 나중 장에서 자세히 다루지만, 여기서는 맛보기로 소개합니다.)

```python
try:
    num1 = float(input("첫 번째 숫자를 입력하세요: "))
    num2 = float(input("두 번째 숫자를 입력하세요: "))
    
    print(f"\n--- 계산 결과 ---")
    print(f"{num1} + {num2} = {num1 + num2}")
    print(f"{num1} - {num2} = {num1 - num2}")
    print(f"{num1} × {num2} = {num1 * num2}")
    print(f"{num1} ÷ {num2} = {num1 / num2:.2f}")

except ValueError:
    print("오류: 숫자를 입력해주세요.")
except ZeroDivisionError:
    print("오류: 0으로 나눌 수 없습니다.")
```
